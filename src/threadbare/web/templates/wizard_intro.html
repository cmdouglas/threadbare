Welcome to Threadbare

This wizard gets your mirror running in a few steps. You'll need access to the Discord Developer Portal for the server you want to mirror.

  1. Create a Discord application and a Bot under it.
  2. Enable the Message Content intent on the Bot tab — miss this and every message body silently arrives empty.
  3. Copy the bot token and the application's client ID (both on the Bot/OAuth2 tabs).
  4. Invite the bot to your server using a generated link (bot scope, View Channels + Read Message History only).
  5. Choose which channels get mirrored.
  6. Under OAuth2, add the redirect URI this wizard shows you, then paste in the client secret to finish.
